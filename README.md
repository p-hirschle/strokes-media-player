# 'Reality Awaits' Media Player 

Um *media player* com estética western/cowboy que referencia o álbum 'Reality Awaits' da banda The Strokes, desenvolvido usando SvelteKit, TypeScript, Tailwind CSS, e Howler.js.    
    
Acesse em: https://realityawaits.vercel.app

## Funcionalidades

- Player de música completo (play/pause, previous/next, volume, select)
- Tracklist com todas as músicas **(apenas as faixas 4 e 6 estão disponíveis até a data de lançamento oficial: 26/06/2026)**
- Visualização das letras das músicas (lyrics)
- Efeitos visuais e sonoros de interação com o usuário
- Design responsivo (mobile e desktop)

## Tecnologias

- **SvelteKit**
- **TypeScript**
- **Tailwind CSS**
- **Howler.js**
